📸 截圖匯出步驟（給 Hank）

方法 A：Chrome / Edge 開發者工具截圖（推薦）

  1. 在 Chrome / Edge 開這個檔案
  2. 按 F12 開啟開發者工具
  3. 按 Ctrl + Shift + M 切換到裝置模擬模式
  4. 上方 dimensions 改成 Responsive，輸入寬 2500 高 843
  5. 右上方 ⋮（三點選單）→ Capture full size screenshot
  6. 得到的 PNG 直接上傳到 LINE OA Manager → Rich Menu

方法 B：直接視窗截圖（簡單但解析度可能不夠）

  1. 把瀏覽器視窗拉到最大
  2. 用 Windows Snipping Tool (Shift + Win + S) 框選下方主視覺
  3. 儲存後用線上工具放大到 2500×843

⚠️ LINE 規格要求：寬 2500 × 高 843（最大版），檔案 1MB 內，PNG / JPG 皆可。

🎨 設計說明（給 Hank / 鈺泰發看）

  • 主色配比：深墨 #1A1A1A（80%）+ 古典金 #B89855（15% 點綴）+ 米白 #FAF7F0（5% 文字）
  • 字體：標題用 Noto Serif TC（思源宋體 / 襯線中字）→ 房地產業專業 + 古典氣質
  • 副標：英文 ALL CAPS + Noto Sans TC → 現代感平衡
  • 圖示：細線條（stroke 1.8）金色 SVG，全部統一風格
  • 圓圈外框：金色 1.5px + 外圈 1px 雙線 → 高級感 / 不過度裝飾
  • 分隔線：中央十字金線（透明度 25%），低調分區
  • 右上角金點：每格獨立性提示，整體節奏統一

為什麼選這個設計方向？ 宜蘭房地產競品（揚京 / 搜博代工）的圖文選單普遍是「卡通 emoji + 鮮豔配色」， 走「**深墨 + 古典金 + 襯線中字**」這條線可以瞬間拉開檔次，呼應鈺泰發 「鈺泰用心、購屋安心、建商放心」的穩重訴求。
